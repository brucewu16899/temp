功能：
  • 对一个 table 中指定的列进行“合并”。
    所谓“合并”，是指把该列中连续多个相同内容的单元格合并为一个单元格。
    用法：
  1. 定义样式表――
    <style type="text/css">
      colgroup.merge { behavior:url(merge.htc); }
    </style>
  2. 在<table>中指定需合并的列――
    <table border="1">
    <colgroup class="merge">
      <col merge="true" width="150">
      <col merge="true" width="150" align="center">
      <col merge="true" width="150" align="right">
      <col width="150">
    </colgroup>
    <tr>
      ……
    </tr>
    </table>
