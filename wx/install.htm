安装“微信实名”工具按钮

微信网页版目前在群聊页面中无法显示“群昵称”，而只能显示账号昵称，这个小工具就是用来解决这个问题的。 按照下面介绍的方法安装本工具之后，在微信群页面点击按钮，即可显示出用户的真实身份。

仅适用于“二七班西师附小”微信群 :)

目前实测只在 Chrome 浏览器下可以正常使用，在 360 安全浏览器、IE8、Firefox 浏览器中无法使用，其它浏览器尚未测试。

可参照 IE 浏览器的操作方法。

Chrome 浏览器

请拖拽链接 到浏览器的“书签工具栏”。
在拖拽之前，将浏览器设置为“总是显示书签栏”（Ctrl+Shift+B）。

火狐（Firefox）浏览器

请拖拽链接 到浏览器的“书签工具栏”。
在拖拽之前，在浏览器的主菜单上勾选“查看-工具栏-书签工具栏”。

Safari 浏览器

请拖拽链接 到浏览器的“书签栏”。
在拖拽之前，将浏览器设置为“显示书签栏”（Ctrl+Shift+B）。

Opera 浏览器

请拖拽链接 到浏览器的“个人栏”。
在拖拽之前，在浏览器的主菜单上勾选“查看-工具栏-个人栏”。
